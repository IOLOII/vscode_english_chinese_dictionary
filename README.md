# EnglishChineseDictionary README

在状态栏中显示选中词或短语的释义

## Features

点击状态栏显示更全信息, 包括词形变化等.

词典数据来源: https://github.com/skywind3000/ECDICT. 由于数据较大, 本机启动环境载入时需几秒. 之后的查询不需再次载入.
